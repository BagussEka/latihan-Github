# Aplikasi Pengolahan Citra Digital

Aplikasi desktop berbasis Python untuk pengolahan citra digital menggunakan Tkinter sebagai antarmuka dan OpenCV sebagai pemrosesan gambar.

## Fitur
- Upload gambar (JPG, PNG, JPEG)
- Grayscale
- Smoothing
- Sharpening
- Emboss
- Informasi gambar (nama file, ukuran, mode warna, operasi)

## Teknologi
Python 3, Tkinter, OpenCV, Pillow (PIL), NumPy

## Instalasi
Clone repository, masuk ke folder project, install library `opencv-python`, `pillow`, dan `numpy`, lalu jalankan file Python utama aplikasi (`python main.py`).

## Penggunaan
Buka aplikasi, upload gambar, pilih operasi pengolahan citra yang diinginkan, dan lihat hasil serta informasi gambar pada panel aplikasi.
